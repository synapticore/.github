## Focus Areas

- **Astronomical Computing** — Large-scale survey analysis, ML classification, interactive 3D visualization
- **Machine Learning** — PyTorch frameworks for ethics, relativity-inspired architectures, and fine-tuning
- **DuckDB Extensions** — Spatial, graph, astronomical, and space-filling curve extensions
- **Scientific Tools** — Paper search, reactive notebooks, AI-assisted UX for researchers
- **Computer Vision** — Open vision encoders, generative models, earth observation
